Hozz létre egy weboldalt, mely a mellékelt API-ról kér le adatokat.
A megkapott adatot jelenítsd meg!

Hozz létre egy menüt, melyben a felhasználó kiválaszthatja, hogy az ID, random vagy egy típus alapján kér le adatot.

Legyen lehetőség megfelelő újabb adat lekérésére is!

Ügyelj a megjelevítésre, hogy tetszetős legyen! :)

mellékelt api:
Endpoints
The following endpoints are available in the API:

Random Joke
This endpoint returns a randomly selected joke.

fetch('https://official-joke-api.appspot.com/jokes/random')
  .then(response => response.json())
  .then(data => console.log(data));
Ten Random Jokes
This endpoint returns ten randomly selected jokes.

fetch('https://official-joke-api.appspot.com/jokes/ten')
  .then(response => response.json())
  .then(data => console.log(data));
Get a Joke by ID
This endpoint returns a joke by its ID.

fetch('https://official-joke-api.appspot.com/jokes/10')
  .then(response => response.json())
  .then(data => console.log(data));
Get a List of Jokes by Type
This endpoint returns a list of jokes by their type.

fetch('https://official-joke-api.appspot.com/jokes/programming/random')
  .then(response => response.json())
  .then(data => console.log(data));
